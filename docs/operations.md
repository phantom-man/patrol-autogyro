# Operational Procedures

## Pre-Flight Checklist

### Aircraft Inspection

- [ ] Airframe integrity check
- [ ] Rotor blades secure, no damage
- [ ] Control surfaces free and correct direction
- [ ] All fasteners tight
- [ ] Fuel tank secure, no leaks
- [ ] Fuel lines connected, no kinks

### Engine Checks

- [ ] Fuel level adequate for mission
- [ ] Oil mix ratio correct (25:1 to 32:1)
- [ ] Spark plug wire connected
- [ ] Propeller secure, balanced
- [ ] Exhaust pipe secure

### Electronics

- [ ] Flight battery charged (>80%)
- [ ] Telemetry link active
- [ ] GPS lock (>8 satellites)
- [ ] RC link solid
- [ ] Kill switch TEST - verify engine stops

### Pre-Takeoff

- [ ] Engine started and warmed up
- [ ] Idle RPM stable (2000-2500 RPM)
- [ ] Throttle response smooth
- [ ] All control inputs verified
- [ ] Mission uploaded (if autonomous)
- [ ] Airspace clear

## Emergency Procedures

### Engine Failure

1. Autogyro will auto-rotate - maintain airspeed
2. Select suitable landing area
3. Flare just before touchdown

### Loss of RC Link

ArduPilot failsafe will activate:
- Short loss: Continue mission or hold
- Long loss: RTL (Return to Launch)

### Fire

1. Activate kill switch immediately
2. Land as soon as safely possible
3. Use fire extinguisher if needed
4. Move away from aircraft

## Post-Flight

- [ ] Engine cooled down before shutdown
- [ ] Fuel valve closed
- [ ] Flight log downloaded
- [ ] Aircraft secured
- [ ] Any issues noted for maintenance
