# Lua Scripts

Optional Lua scripts for ArduPilot.

## Planned Scripts

- Engine health monitoring
- Automatic idle adjustment
- Patrol pattern automation

## Installation

1. Copy `.lua` files to SD card `/APM/scripts/` folder
2. Enable scripting: `SCR_ENABLE = 1`
3. Reboot flight controller
